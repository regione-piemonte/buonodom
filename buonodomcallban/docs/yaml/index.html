get /servizio-attivo
(servizioAttivoGet)
Restituisce informazioni sull'ambiente su cui sta girando il back-end. Es. variabili ambiente, header http ricevuti ecc.

Return type

info_servizio

Example data

Content-Type: application/json
{
  "descrizione" : "descrizione",
  "data" : "2000-01-23",
  "nome" : "nome",
  "servizio_attivo" : true
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Restituisce un oggetto JSON contenente i dati dell'env su cui sta girando il servizio info_servizio
